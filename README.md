# ent-graphql-example

The code for https://entgo.io/docs/tutorial-setup

## Installation

```console
git clone git@github.com:a8m/ent-graphql-example.git
cd ent-graphql-example
go run ./cmd/todo/
```

Then, open [localhost:8081](http://localhost:8081) to see the GraphQL playground.
